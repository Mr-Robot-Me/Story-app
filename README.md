# Story-app

This is an application for a simple blog or posts

Clone me !

$ bundle install

$ rake db:migrate

$ rails s

Open webrick server at localhost:3000

Create an account

Create posts

Deploy to Heroku

*remmember to $ RAIL_ENV=production rake assets:precompile *

If you run into any issues Google around.
